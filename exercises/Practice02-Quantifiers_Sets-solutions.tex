\begin{Ans}{18}
      If $R$ is the set of red cards and $F$ is the set of face cards, we want to find $|R \cup F|$.  This is not simply $|R| + |F|$ because there are 6 cards which are both red and a face card; $|R \cap F| = 6$.  We find $|R \cup F| = 32$.
