We write \(f:X \to Y\) to describe a function with name \(f\text{,}\) domain \(X\) and codomain \(Y\text{.}\) This does not tell us which function \(f\) is though. To define the function, we must describe the rule. Often ths is done with a formula (for example, \(f(x) = x^2\) says that each element of the domain is mapped to its square), or in words (like how we just described the squaring function). You could also define a function with a table or a graph.
